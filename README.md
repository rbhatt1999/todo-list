## 💻 Getting Started with todo-list <a name="getting-started"></a>

### Setup

Clone this [repository](https://github.com/rbhatt1999/todo-list.git) to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:rbhatt1999/todo-list.git 
  cd todo-list
```

### Install

Install this project with:

```sh
  npm install
```

### Start

To start the application, run the following command

```sh
  npm start
```

### Run tests

To run tests, run the following command:

```sh
  npm test
```
